# data/

Place your Church Father text files here in subdirectories named by father ID.

## Structure

```
data/
  augustine/
    confessions.txt
    city-of-god.txt
    tractates-on-john.txt
  chrysostom/
    homilies-on-matthew.txt
    homilies-on-romans.txt
  origen/
    commentary-on-john.txt
    on-first-principles.txt
  jerome/
    commentary-on-matthew.txt
    letters.txt
  cyril/
    commentary-on-john.txt
  ambrose/
    hexameron.txt
    on-the-duties.txt
  evagrius/
    praktikos.txt
    on-prayer.txt
```

## Where to get texts (all free)

- **CCEL** — ccel.org/fathers — best source, clean plain text downloads
- **Tertullian.org** — broad early church library
- **New Advent** — newadvent.org/fathers — copy-paste into .txt
- **Internet Archive** — archive.org — older translations, may need cleanup

## Cleaning tips

Before ingesting, remove:
- Page numbers
- Running headers/footers
- Footnote numbers (keep footnote text if it's substantive)
- Publisher boilerplate

The cleaner the text, the better the retrieval quality.
